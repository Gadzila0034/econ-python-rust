# ?? Статистический анализатор с Rust бэкендом 
 
Проект для статистического анализа данных с высокопроизводительным Rust бэкендом и Python GUI. 
 
## ?? Быстрый запуск 
 
```bash 
# 1. Установите Rust модуль 
cd rust-core 
pip install maturin 
maturin develop 
cd .. 
 
# 2. Установите Python зависимости 
cd statistics_app 
pip install -r requirements.txt 
cd .. 
 
# 3. Запустите приложение 
python statistics_app/main.py 
```
